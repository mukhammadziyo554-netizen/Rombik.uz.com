# Rombik.uz Ecommerce (UZ/RU)

Modern, mobile-first ecommerce layout with instant UZ/RU language switching.

## Pages
- `/` Home
- `/category/` Category
- `/product/` Product
- `/cart/` Cart
- `/checkout/` Checkout
- `/checkout/confirmation.html` Confirmation
- `/delivery/` Delivery & Payment
- `/about/` About
- `/contact/` Contact

## Run locally
Serve with any static server. Example (Python):

```zsh
python -m http.server 8000
```

Open: http://localhost:8000
